# 正则表达式可视化调试器

NFA 状态机可视化、逐步匹配高亮、分组捕获标注、回溯追踪的交互式正则调试工具。

## 技术栈

- Vue 3 + TypeScript + Vite
- Pinia 状态管理
- Tailwind CSS 样式
- Canvas 2D 状态机渲染

## 功能

- 正则表达式 NFA 状态机 Canvas 可视化
- 逐步匹配高亮（字符→状态→转移边）
- 分组捕获实时标注与颜色编码
- 回溯路径追踪与步骤计数
- 20+ 常用模式模板库
- 正则语法树分层可视化
- 匹配性能统计（步骤数/回溯次数/耗时）
- 反向引用/零宽断言/贪婪惰性模式支持

## 启动

```bash
cd frontend
npm install
npm run dev
```
